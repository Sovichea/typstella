# Complex-script typography

Purpose: inspect shaping and fallback font coverage for Khmer, Lao, Devanagari,
Bengali, and Myanmar. Rendering a script is not the same as providing deep
editing, segmentation, spelling, or completion support.

Choose fonts with appropriate OpenType shaping coverage if the named defaults
are unavailable. Typsastra never downloads or exports a font automatically.

Tutorial: <https://github.com/Sovichea/typsastra/blob/main/docs/tutorials/DOCUMENT_TYPOGRAPHY.md>
