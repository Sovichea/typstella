# Multilingual article

This is the combined v0.5.x project example. Set `main.typ` as the main file,
then edit the included sections.

It demonstrates primary and embedded-script typography, English/Khmer/Arabic
disjoint-script content, explicit English/French/Spanish same-script scopes,
named and anonymous blocks, unavailable-provider warnings, terminology, and
keyboard-language completion without changing spellcheck ownership.

Install optional Arabic, French, and Spanish providers to enable those scopes.
Their absence must not prevent the document from compiling.

Tutorial: <https://github.com/Sovichea/typsastra/blob/main/docs/tutorials/MULTILINGUAL_SPELLCHECK.md>
