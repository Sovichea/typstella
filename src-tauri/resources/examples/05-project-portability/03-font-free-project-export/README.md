# Font-free project export

Export this folder and inspect the resulting archive. It should contain source
and applicable workspace settings, but no font binary, generated PDF, cache,
symlink, `.git`, `node_modules`, or build directory.

Recipients must install required fonts separately. This policy applies even to
fonts whose licenses might otherwise permit redistribution.

Tutorial: <https://github.com/Sovichea/typsastra/blob/main/docs/tutorials/PROJECT_IMPORT_AND_EXPORT.md>
