# Khmer segmentation comparison

Compare ordinary Typst justification, Typsastra's experimental render
preparation, and the recommended tracking-tuned layout. Render preparation is
optional and default-off; use this fixture to evaluate it, not as a claim that
source transformation is required for Khmer writing.
