# Khmer deep support

This document exercises the reference deep-support provider: Khmer-aware word
boundaries, spellcheck, completion, and script editing behavior. Khmer is
bundled, so no provider download is required. Install a suitable Khmer font if
the configured family is unavailable.

Expected behavior includes grapheme-safe cursor/deletion operations and Khmer
word selection at the beginning of a line.

Reference: <https://github.com/Sovichea/typsastra/blob/main/docs/KHMER_SPELLCHECK.md>
